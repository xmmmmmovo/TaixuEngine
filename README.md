## How to build

**Prerequisites:**

- C++20 compliant compiler
- [python](https://www.python.org/)
- [cmake](https://cmake.org/)
- [Vulkan SDK](https://vulkan.lunarg.com/sdk/home)

**Build:**

```shell
mkdir build
cd build
cmake ..
make
```
